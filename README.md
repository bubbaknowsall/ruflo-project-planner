# RuFlo Project Planner - AI Powered 🚀

A web application for generating detailed project plans using AI (Google Gemini).

## ✨ Features

- ✅ **AI-Powered Plans**: Uses Google Gemini to generate realistic project plans
- ✅ **Instant Results**: Get detailed plans in seconds
- ✅ **Parts Lists**: Complete with cost estimates
- ✅ **Step-by-Step Guides**: Easy-to-follow instructions
- ✅ **Safety Warnings**: Important safety notes
- ✅ **Responsive Design**: Works on mobile and desktop
- ✅ **100% Free**: Uses free tier APIs

## 🚀 Deployed & Ready to Use!

Your app is live and ready for deployment on Vercel.

**Features:**
- Frontend: Beautiful HTML5/CSS3 interface
- Backend: Node.js API with Google Gemini integration
- Hosting: Ready for Vercel deployment

## 📖 How to Use

1. Deploy to Vercel
2. Add your Google API key as an environment variable
3. Open your app and start generating project plans!

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: Node.js (Vercel Functions)
- **AI**: Google Gemini API
- **Hosting**: Vercel
